# Emerald Isle

## Abstract

現在開いているタブの `title` と URL を Plain Text, HTML, Markdown の3種のフォーマットのいずれかでクリップボードにコピーできる Chrome拡張機能 です。

## Background

元々 Create Link を利用していましたが、 Chrome `96.0.4664.110` にアップデートしたところ挙動が不安定になってしまいました。

※`5.1`, `5.5` の2つを試験していずれも挙動不安定

現象としては、右クリックから Create Link のいずれの項目も選択時にコピーできたりできなかったりと疎らな反応になってしまった、という状態です。

この状態が不便だったので、同等の機能のものを自作した次第。
